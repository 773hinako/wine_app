# ワイン記録アプリ - 使い方ガイド

このアプリはPWA（Progressive Web App）として動作するワイン記録アプリです。
完全にオフラインで動作し、データは全てブラウザに保存されます。

## インストール方法

### 必要なもの
- Python 3（ほとんどのPCに標準でインストール済み）
- または、Node.js

### 手順

1. このフォルダをPCの好きな場所に配置

2. コマンドプロンプト（またはターミナル）を開く
   - Windows: フォルダ内で Shift + 右クリック → "PowerShellウィンドウをここで開く"
   - Mac: フォルダを右クリック → "サービス" → "フォルダに新規ターミナル"

3. サーバーを起動

   Pythonの場合:
   ```
   python -m http.server 8000
   ```

   Node.jsの場合（事前にインストール必要: npm install -g http-server）:
   ```
   http-server -p 8000
   ```

4. ブラウザで以下のURLを開く
   ```
   http://localhost:8000/public/
   ```

5. PWAとしてインストール（任意）
   - ブラウザのアドレスバー右端の「インストール」ボタンをクリック
   - アプリとして独立したウィンドウで使えるようになります

## 機能

✅ ワインの記録（名前、産地、品種、ヴィンテージなど）
✅ 詳細なテイスティングメモ（色、香り、味わい、余韻）
✅ 写真撮影・添付（自動圧縮）
✅ 評価（5段階星評価）
✅ お気に入り登録
✅ ソート・フィルター機能
✅ 統計ダッシュボード
✅ ダークモード
✅ データエクスポート/インポート（JSON）
✅ 完全オフライン動作

## データのバックアップ

1. アプリ内の「📤」ボタンをクリック
2. JSONファイルがダウンロードされます
3. このファイルを保存しておけば、いつでも復元可能

## サーバーの停止方法

コマンドプロンプトで Ctrl+C を押す

## 問題が起きたら

- ブラウザのキャッシュをクリア
- Ctrl+Shift+R (Mac: Cmd+Shift+R) でハードリフレッシュ
- ポート8000が使えない場合は、別の番号（例: 3000）を使う
